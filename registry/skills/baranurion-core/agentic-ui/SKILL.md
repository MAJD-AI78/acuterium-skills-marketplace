# Agentic Ui

## Description
You are designing AI-native interfaces for a 2026+ world where every app is AI-powered. This skill provides production-ready patterns for intent preview (showing users what AI will do before execution), autonomy controls (letting users choose between manual/supervised/autonomous AI actions), task decomposition UI (breaking goals into steps), streaming states, tool call transparency, confidence indicators, and error recovery.
